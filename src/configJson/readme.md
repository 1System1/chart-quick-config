#### json 格式

```
{

    customConfig:{}, // 自定义配置
    keyMap:{}, // 字段映射key

    icon:"",// 图标
    isDefault:Boolean,// 系统默认
    extendConfig:{},// 可以扩展的配置
    editSlot:"",//编辑配置组件
}
```
